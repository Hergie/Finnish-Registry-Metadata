- **Source:** UTH-kyselytutkimus

## Description

Haastatteluaineisto ulkomaalaistaustaisista henkilöistä vuodelta 2014. N = 3262. UTH-kyselyaineisto sisältää työhön, koulutukseen, terveyteen ja hyvinvointiin liittyviä tietoja. A-osioon on liitetty joitakin rekisteritietoja sekä rekisterin ja kyselyn pohjalta tehtyjä muuttujia, jotka liittyvät olennaisesti maahanmuuttajiin. Tällaisia ovat esimerkiksi sukupuoli, ikä, kansalaisuus, taustamaa, vanhempien syntymämaat, maahan- ja maastamuuttopäivät, asuinvuodet Suomessa sekä ikä muuttovuonna. Aineistoa on kuvattu tarkemmin <a href=http://stat.fi/tup/maahanmuutto/uth-tutkimus/menetelmaseloste.html> menetelmäselosteessa. </a> Esimerkkejä aineistosta aiemmin julkaistuista tuloksista löydät <a href=http://stat.fi/tup/maahanmuutto/uth-tutkimus.html> täältä. </a> Lisätietoa löytyy myös <a href=https://www.thl.fi/fi/tutkimus-ja-asiantuntijatyo/hankkeet-ja-ohjelmat/uth-ulkomaista-syntyperaa-olevien-tyo-ja-hyvinvointi-tutkimus> THL:n sivuilta. </a>

Kaikki osiot on saatavilla Tilastokeskuksen käyttölupahakemuksella etäkäyttöön. Tässä kuvauksessa on kuvattuna osiot D-N. Koulutukseen ja työmarkkinoihin liittyvien muuttujien kuvaus löytyy otsikon <a href=https://taika.stat.fi/fi/aineistokuvaus.html#!?dataid=UTH_2014_jua_TK17_001.xml> "UTH-aineisto 2014 (koulutus ja työmarkkinat, Tilastokeskus)" </a> alta.

<b>Käyttölupaprosessi</b>
Mikäli käyttöön haetaan tämän kuvauksen osioita D-N (terveys ja hyvinvointi), on Tilastokeskuksen käyttölupaprosessissa muutamia poikkeuksia. Käyttölupahakemus ja tutkimussuunnitelma tulee toimittaa Tilastokeskuksen kirjaamon lisäksi THL:ään arvioitavaksi osoitteeseen tuija.jaaskelainen@thl.fi. Lisääthän tällöin tutkimussuunnitelmaan tiedon, mitä muuttujia aiot tutkimuksessasi käyttää. THL arvioi hakemuksen ja pyytää täydennyksiä hakijalta, mikäli katsoo sen tarpeelliseksi. Käyttölupa voidaan tarvittaessa myöntää erikseen TK:n osiolle (A-C) ja THL:n osioille (D-N).

<b>Verrokkiaineistot </b>
Koulutus- ja työmarkkinaosioille on saatavilla verrokkiryhmä Työvoimatutkimuksesta (LFS) ja terveys- ja hyvinvointiosiolle Alueellisesta terveys- ja hyvinvointitutkimuksesta (ATH), jota on viime vuosina kutsuttu myös <a href=https://www.thl.fi/fi/tutkimus-ja-asiantuntijatyo/vaestotutkimukset/aikuisten-terveys-hyvinvointi-ja-palvelututkimus-ath>  Aikuisten terveys-, hyvinvointi- ja palvelututkimukseksi. </a>.  ATH-verrokkijoukolta ei ole kysytty kaikkia samoja kysymyksiä kuin UTH-kohderyhmältä. Kaikki muuttujat puuttuvat verrokkiryhmältä blokeissa G ja N.
